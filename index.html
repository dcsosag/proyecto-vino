Predicción de Calidad del Vino

Machine Learning con Databricks, XGBoost y MLflow

Resumen del Proyecto

Este proyecto utiliza datos fisicoquímicos del vino portugués "Vinho Verde" para predecir si un vino será de alta calidad (puntuación ≥ 7). Implementé un flujo completo de MLOps utilizando Databricks y MLflow.

#Python #XGBoost #MLflow #Spark #Databricks

1. Análisis de Datos (EDA)

Identificamos que las variables más influyentes son:

  • Alcohol: Los vinos de alta calidad presentan mayor graduación alcohólica.
  • Densidad: Existe una relación inversa con la calidad.

[Aquí puedes insertar la imagen de tus Box Plots]

2. Entrenamiento y Optimización

Modelo Base: Random Forest
Entrenamos un clasificador básico obteniendo un AUC de 0.854. Se utilizó una clase personalizada para manejar las probabilidades de predicción.
Optimización: XGBoost + Hyperopt
Usando búsqueda de hiperparámetros en paralelo, logramos elevar el AUC a 0.904. Ajustamos: learning_rate, max_depth y reg_lambda.

3. MLOps con MLflow Model Registry

El modelo no solo se entrenó, se gestionó profesionalmente:

Tracking

Registro de métricas AUC y parámetros.

Registry

Control de versiones (V1 a V3).

Production

Transición de modelo a estado "Production".

4. Inferencia a Escala

Finalmente, registramos el modelo como una Spark UDF. Esto permite realizar predicciones sobre tablas Delta con millones de filas de forma distribuida. 

# Ejemplo de carga de modelo en producción
model_uri = "models:/wine_quality/Production"
predict_udf = mlflow.pyfunc.spark_udf(spark, model_uri)
